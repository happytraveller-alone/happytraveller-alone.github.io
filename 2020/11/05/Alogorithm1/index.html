1.How the Greedy Method Works

1.1 Optimization problems

渴婴问题
假设有一个口渴但很聪明的婴儿，他可以喝一杯水，一箱牛奶等，总共n种液体。令$a_i$为第i种液体可用的盎司量。根据她对口味和对营养的渴望的经验，她还为第i种液体分配了一些令人满意的因子$S_i$。
问题：如果婴儿需要喝几盎司的液体，她应该喝多少液体？
令$x_i(1\leq i\leq n)$为婴儿将要喝的第i种液体的量，如果未加载容器i，则为0，在其他情况下为1。
我们期望得到$\max\limits_{x_i{1\leq i \leq n }} {\sum_{i=1}^n x_is_i}$ 实际的状态分析为$$\begin{cases}max{\sum_{i=1}^n x_i=t }, \ 0 \leq x_i \leq a_i \end{cases}$$

装货问题

一艘大船要装载货物。 货物已装箱，所有集装箱的尺寸均相同。 不同的容器可能具有不同的重量。令$w_i(1\leq i \leq n)$为第i个容器的重量。该船的载货量为c。 问题：我们希望以最大数量的集装箱装船。 我们期望得到$max\{\sum_{i=1}^n x_i\}$ 实际的状态分析为$$\begin{cases}\sum_{i=1}^n x_iw_i \leq c, \\ x_i \in\{0,1\} \end{cases}$$ 优化问题是从所有可行的解决方案中找到最佳解决方案的问题。它由三个部分组成 一组决策变量：$x_i(1 \leq i \leq n)$目标函数$f(x_1，\cdots，x_n)$；一组约束$g_i(x_1，\cdots，x_n)\leq 0( 1 \leq i \leq m)$；可行解：满足约束的实例最优解：优化(最大/最小)目标函数的实例
